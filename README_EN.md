# we-chat-devtools
# 微信开发者工具小程序上传封装SDK（PHP）

Instruct WeChat devtools to perform operations such as login, preview, and upload.



## Requirements

 - PHP >= 7.0.0
 - Laravel >= 5.5.0
    

## Installation
