# we-chat-devtools
# 微信开发者工具小程序上传封装SDK（PHP）

指示微信开发工具进行登录、预览、上传等操作。



## 需求

 - PHP >= 7.0.0
 - Laravel >= 5.5.0
    

## 安装

通过 Composer 安装 we-chat-devtools

`composer require li-bao-jia/we-chat-devtools`
    
或编辑 composer.json 文件安装
    
    ```js
    
        {
            "require": {
                "li-bao-jia/we-chat-devtools": "1.*"
            }
        }
    
    ```
